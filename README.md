# saiganesh-portfolio
This is my personal portfolio website designed to highlight my professional journey as a QA Engineer and Software Developer.
The site showcases my skills, real-time project experience, certifications, and education in a clean and responsive layout.

Built using HTML, Tailwind CSS, and JavaScript, the portfolio includes features such as dark mode, interactive project cards, smooth scrolling, and a downloadable resume.
Deployed using GitHub Pages.
